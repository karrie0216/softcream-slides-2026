## ワークシートの使い方

| WORK | 区分 | 時間 | 内容 |
|---|---|---|---|
| **WORK 01** | チーム | 30秒 | 金口（絞り口）の形から動物パーツを連想してチームで書き留める |
| **WORK 02** | チーム | 5分 | 機械・条件の疑問を付箋に書いてチームから1問質問 → 回答を記録 |
| **WORK 03** | チーム | 10分 | ペルソナシート：名前・年齢・来園目的・カフェに入る理由・食後の気持ち |
| **WORK 04** | **個人** | 10分 | 1人1案のアイデアスケッチ＋コンセプト一言 |

## ファシリテーターメモ

| よくあるつまずき | 対処法 |
|---|---|
| Q&Aで時間が溶ける | 付箋1枚/チームを回収してまとめて回答。5〜6問で打ち切る |
| ペルソナが「誰でも向け」になる | 「名前をつけてみて」「その人のLINE名は？」と具体化を促す |
| 水族館・他の動物園の話が混ざる | 「今日は**福岡市動物公園だけ**で考えよう」とやさしく軌道修正 |
| 「絵が下手で書けない」 | 「丸・三角・棒線でOK。伝わればいい」と繰り返す |
| 早く進んだ場合 | WORK 04のスケッチ時間を延ばして完成度を上げる |
| 遅れた場合 | 事例を2本に絞る（旭山動物園を省略）。WORK 04は「動物を選ぶだけ」で終わらせ、スケッチは宿題に |
